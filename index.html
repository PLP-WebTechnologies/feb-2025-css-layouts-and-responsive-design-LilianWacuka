Flexbox
Flexbox is a one-dimensional layout method for laying out items in a row or column.
It is a powerful tool for creating responsive layouts and aligning items within a container.
Grid.
Grid is a two-dimensional layout method for laying out items in rows and columns.
It is a powerful tool for creating complex layouts and aligning items within a container.
Float.
Float is a layout method that allows elements to be positioned to the left or right of their container.
It is an older method of layout and is not as flexible as flexbox or grid.
Position.
Position is a layout method that allows elements to be positioned relative to their container or the viewport.
It is a powerful tool for creating complex layouts and aligning items within a container.
It is often used in conjunction with other layout methods such as flexbox and grid.
